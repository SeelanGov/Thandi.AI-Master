# THANDI.AI Feedback Form Template

**Purpose:** Capture edge cases and system failures to improve accuracy

---

## Google Form Structure

### Section 1: Basic Information

**School Name:** [Text field]

**Teacher Name:** [Text field]

**Date:** [Date picker]

**Grade Level:** [Dropdown: 9, 10, 11, 12]

---

### Section 2: The Query

**What did the student ask?** [Long text field]
*Please write the exact question as the student asked it*

**What was the student actually trying to find out?** [Long text field]
*In your own words, what did they really need?*

---

### Section 3: System Response

**What did THANDI.AI say?** [Long text field]
*Copy-paste or summarize the system's response*

**Was the response helpful?** [Multiple choice]
- [ ] Very helpful - Student got exactly what they needed
- [ ] Somewhat helpful - Partially answered the question
- [ ] Not helpful - Missed the point
- [ ] Confusing - Made things worse
- [ ] Wrong - Factually incorrect

---

### Section 4: The Problem

**What went wrong?** [Checkboxes - select all that apply]
- [ ] System didn't understand the question
- [ ] Suggested wrong careers
- [ ] Missed important constraints (e.g., "no matric")
- [ ] Gave outdated information
- [ ] Too complex/technical language
- [ ] Didn't acknowledge conflicting goals
- [ ] Hallucinated information (made things up)
- [ ] Too slow to respond
- [ ] Other: [Text field]

**Specific issue:** [Long text field]
*Describe exactly what was wrong*

---

### Section 5: What Should Have Happened

**What would have been the correct answer?** [Long text field]
*Based on your expertise, what should the system have said?*

**Which careers should have been suggested?** [Text field]
*List the careers that would have been appropriate*

---

### Section 6: Impact

**How did this affect the student?** [Multiple choice]
- [ ] Minor - Student wasn't bothered
- [ ] Moderate - Student was confused but we fixed it
- [ ] Significant - Student lost confidence in the system
- [ ] Severe - Student made a wrong decision based on this

**Did you have to intervene?** [Multiple choice]
- [ ] Yes - I corrected the information immediately
- [ ] Yes - I had to do additional research
- [ ] No - Student figured it out themselves
- [ ] No - We just moved on

---

### Section 7: Pattern Recognition

**Have you seen this type of error before?** [Multiple choice]
- [ ] Yes - This happens frequently
- [ ] Yes - I've seen it once or twice
- [ ] No - This is the first time

**If yes, how often?** [Text field]

---

### Section 8: Suggestions

**How could the system be improved?** [Long text field]
*Your ideas for fixing this issue*

**What additional careers should we add?** [Text field]
*Careers students ask about that aren't covered*

**What additional features would help?** [Long text field]
*E.g., province-specific info, disability accommodations, etc.*

---

### Section 9: Follow-Up

**Can we contact you for more details?** [Multiple choice]
- [ ] Yes - Email me
- [ ] Yes - Call me
- [ ] No - This form is sufficient

**Contact Email:** [Text field - optional]

**Contact Phone:** [Text field - optional]

**Best time to reach you:** [Text field - optional]

## Internal Processing (Not in Form)

### Priority Classification:
- **P0 (Critical):** Factually wrong information, hallucinations
- **P1 (High):** Missed major constraints, wrong career suggestions
- **P2 (Medium):** Confusing language, missing features
- **P3 (Low):** Minor improvements, nice-to-haves

### Response Timeline:
- P0: Fix within 24 hours, notify teacher
- P1: Fix within 1 week, include in next update
- P2: Fix within 1 month, batch with similar issues
- P3: Consider for future roadmap

### Tracking Metrics:
- Feedback submissions per school
- Most common error types
- Careers most frequently requested
- Teacher satisfaction scores
- Student impact severity
